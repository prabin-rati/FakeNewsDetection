# 🛡️ Fake News Detection System
[![Python Quality Check](https://github.com/prabin-rati/FakeNewsDetection/actions/workflows/python-lint.yml/badge.svg)](https://github.com/prabin-rati/FakeNewsDetection/actions)

A Machine Learning based web application designed to identify and classify fake news articles. Developed as part of the **BCA 6th Semester Project (TU)**.

---

## 🚀 Features
* **Machine Learning Model:** Uses Passive Aggressive Classifier / Logistic Regression for high-accuracy detection.
* **Vectorization:** Implements `TfidfVectorizer` for natural language processing.
* **Web Interface:** Built with Flask/Streamlit for easy user interaction.
* **CI/CD Integrated:** Automated code quality checks using GitHub Actions.

## 🛠️ CASE Tools & Technology Stack
* **IDE:** Visual Studio Code (VS Code)
* **Version Control:** Git & GitHub
* **Automation:** GitHub Actions (Linting)
* **Language:** Python 3.9+
* **Libraries:** Scikit-learn, Pandas, NumPy, Flask
